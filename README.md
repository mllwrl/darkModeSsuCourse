# 🌙 SSU Course Dark Mode
**Тёмная тема и стиль Dracula для портала `course.sgu.ru`**

Устал от яркого белого фона при подготовке к парам ночью? Это расширение исправляет дизайн СГУ, делая его тёмным. Выбирай между классическим тёмным режимом или Dracula.

---

## 🛠 Установка

### 🔴 Яндекс.Браузер (Самый простой способ)
1. Скачай файл `darkModeSsuCourse.crx` из папки `updates`.
2. Открой страницу дополнений, введя в адресную строку: `browser://extensions/`
3. Включи **"Режим разработчика"** (справа вверху).
4. Просто **перетащи файл .crx** в окно браузера.
5. Подтверди установку. Готово!

---

### 🟡 Google Chrome
1. Скачай архив `darkModeSsuCourse.zip` из папки `updates` и **распакуй его** в отдельную папку.
2. Перейди по адресу: `chrome://extensions/`
3. Включи **"Режим разработчика"** (Developer mode).
4. Нажми кнопку **"Загрузить распакованное расширение"** (Load unpacked).
5. Выбери папку, в которую ты распаковал архив.

---

### 🟠 Mozilla Firefox
*На данный момент расширение находится на проверке. Пока доступна временная установка:*

1. Скачай файлы расширения (или архив и распакуй его).
2. В адресной строке введи: `about:debugging#/runtime/this-firefox`
3. Нажми кнопку **"Load Temporary Add-on..."** (Загрузить временное дополнение).
4. Выбери файл `manifest.json` в папке расширения.
> **Важно:** Временное расширение удалится после перезапуска браузера. Ждём одобрения официального аддона!

---

## 🚀 Как использовать
1. Зайди на [course.sgu.ru](https://course.sgu.ru).
2. Нажми на иконку расширения в панели браузера
3. Выбери одну из тем:
   * **Dark** — глубокий черный.
   * **Dracula** — фиолетовые тона.
   * **Светлая** — возврат к заводским настройкам.
